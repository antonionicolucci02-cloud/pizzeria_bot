pizzeria_bot - pacchetto pronto (python-telegram-bot v20+)

Istruzioni rapide:
1) Scarica e scompatta la cartella pizzeria_bot
2) Apri Terminale e vai nella cartella:
   cd ~/Downloads/pizzeria_bot   (o dove l'hai scompattata)
3) Crea ed attiva l'ambiente virtuale:
   python3 -m venv venv
   source venv/bin/activate
4) Installa dipendenze:
   pip install python-telegram-bot==20.7
5) Apri main.py e inserisci il tuo TOKEN Telegram nella variabile TOKEN
6) Avvia:
   python3 main.py
7) Su Telegram invia /start e poi /menu
